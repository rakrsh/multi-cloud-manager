# Multi-Cloud Manager

Welcome to the Multi-Cloud Manager documentation. This project provides a centralized, monitoring dashboard that aggregates metadata and status information from AWS, Azure, GCP, and Salesforce.

## Key Features

- **Single Pane of Glass:** Unified view for infrastructure and SaaS health.
- **Provider Adapter Pattern:** Normalized vendor-specific data.
- **High Concurrency:** Parallel data fetching using Go routines.
- **Glassmorphism UI:** Modern, high-density data visualization.
